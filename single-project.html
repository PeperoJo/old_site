GE Align Lighting Technology

"color-tunable LED light bulb capable of producing both the effects of the GE Align AM and PM bulbs"

For the GE Lightingâ€”Align Lighting Technology project, Group 8 was tasked with developing a model and prototype for a single energy-efficient, color-tunable LED light bulb capable of producing both the energizing effect of the GE Align AM bulb and the soothing effect of the GE Align PM bulb. A combined AM/PM bulb is easier for customers to install and use, and enables them to manage their circadian rhythms more effectively.

The invigorating and calming capabilities of the GE Align AM and PM bulbs, respectively, are achieved by emitting certain hues of light: blue-white in the case of the AM bulb, and yellow-toned for the PM bulb. Due to the color-tunable nature of the combined capability GE light bulb, users will also be able to change the color of the light bulb based on data from a CIE color space provided by GE. Customers will be able to interact with the program through a graphical user interface, or GUI.

  • Collaborated with team members to develop an LED light bulb that could change color based on the time of day or the color preference of the user
  • Developed a code and graphical user interface (GUI) using MATLAB software which allowed the user to select and change the color of the light bulb
  • Designed a circuit using an Arduino, breadboard, transistors, and resistors that allowed the MATLAB program to communicate with the color-tunable light bulb
